Kombinationsregeln
  • Element + Zusatztext darunter: 40px Abstand, 1–3 Absätze, untereinander oder mehrspaltig.
  • Element + Subline darüber: Subline steht in der Kopfzone (.slide-head), nicht im Element — funktioniert mit jedem Element.
  • Rings/Ring+Value/Value-Row verteilen sich über Spalten: 2 Elemente → 2-spaltig, 3 → 3-spaltig, 4 → 4-spaltig — gleiche Logik wie Steps.
  • Notation für Slide-Briefings: Layout · Element, z. B. L6 · E8 + Copy oder L4 · E1 oder L3 · E3.
